# TV 直播应用开发计划

**项目名称**: Mini TV Live（辉哥定制版）
**创建时间**: 2026-03-21 23:28
**目标体积**: < 10MB
**平台**: Android (TV/手机/平板通用)

## 功能范围

### 核心功能 (MVP)
- [x] 频道列表展示（支持 JSON 配置）
- [x] 视频播放器（HLS/M3U8 支持）
- [x] 频道切换
- [x] 遥控器支持（TV 方向键）
- [x] 全屏播放
- [x] 收藏功能
- [x] 分组导航

### UI 设计
- [x] 极简深色主题
- [x] 大卡片布局
- [x] 分组标签导航
- [x] 无干扰播放界面
- [x] 加载/错误状态提示

### 直播源配置
- [x] 爱小鹿 TV 源 (46 个频道)
- [x] 央视全套 (20 个)
- [x] 主流卫视 (26 个)
- [x] 备用源说明文档

## 技术选型

- **框架**: Flutter 3.x
- **播放器**: video_player + chewie
- **HTTP**: dio
- **状态管理**: provider
- **TV 适配**: 硬件键盘事件处理

## 体积优化策略

1. 只保留必要依赖
2. 使用 Flutter tree shaking
3. ProGuard 代码混淆
4. ABI 分包构建
5. 移除冗余资源
